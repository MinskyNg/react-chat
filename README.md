# 基于Node.js、Socket.io、React的实时聊天室

[react-chat](http://minsky.me/chat)

## How to start

npm install & npm run dev

## Features

* 用户注册，登录登出
* 创建群组，群聊私聊
* 个人资料修改，自定义头像
* 提供用户和群组搜索
* 设置私聊屏蔽、声音提醒、桌面提醒
* 支持图片上传（感谢sm.ms提供图床）
* 接入图灵机器人对话API

## Tech Stack

* Express
* Socket.io
* React
* Redux
* React-Router
* Immutable.js
* Isomorphic-fetch
